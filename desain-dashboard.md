# Style Guide: Desain Dashboard Admin 

## 1. Tujuan Desain
Dokumen ini berfokus pada estetika visual dashboard admin tanpa mengandung isi konten atau data fungsional. Tujuannya adalah menjadi acuan tampilan yang rapi, modern, dan konsisten untuk area admin panel.

## 2. Nuansa Visual
- Modern
- Bersih
- Profesional
- Ringan namun elegan
- Terlihat tegas dan nyaman dipakai

## 3. Warna Utama
- Accent utama: #e92c1e
- Background utama: putih dan abu-abu sangat muda
- Teks utama: hitam gelap dan abu-abu tua
- Border: abu-abu muda
- Highlight interaktif: accent dengan transisi halus

## 4. Bentuk dan Komposisi
- Sudut elemen dibuat membulat
- Kartu diberi jarak yang cukup agar tidak terasa padat
- Layout dibuat simetris dan teratur
- Elemen utama ditampilkan dengan hierarki visual yang jelas

## 5. Elemen Visual
- Bayangan lembut untuk memberi kedalaman
- Efek glassmorphism pada panel kartu
- Ikon sederhana, modern, dan konsisten
- Spasi antar elemen dibuat proporsional

## 6. Tipografi
- Gunakan font modern dan mudah dibaca
- Heading tegas dan ringkas
- Label kecil diberi uppercase dengan tracking yang rapih
- Ukuran teks dibuat proporsional untuk hierarki visual

## 7. Komponen Utama
### Header
- Menampilkan identitas visual area admin
- Terlihat bersih dan tidak terlalu ramai
- Dapat diberi elemen sapaan atau status ringan

### Kartu Panel
- Berbentuk rounded card
- Latar putih/soft dengan border tipis
- Dilengkapi shadow halus
- Cocok untuk menampilkan bagian penting secara ringkas

### Sidebar
- Terstruktur dan rapi
- Item menu diberi jarak yang konsisten
- Menu aktif ditandai dengan warna dan bobot teks yang lebih tegas

### Tombol Aksi
- Bentuk rounded
- Warna accent untuk aksi utama
- Hover state halus dan tidak terlalu mencolok

## 8. Interaksi Visual
- Transisi antar elemen dibuat halus
- Hover state ringan dan elegan
- Tidak ada efek yang terlalu ramai
- Fokus visual tetap jelas saat pengguna berinteraksi

## 9. Responsivitas Visual
- Tampilan desktop terasa lega dan terstruktur
- Tampilan tablet tetap rapi dan seimbang
- Tampilan mobile disusun secara sederhana agar tetap nyaman dipakai

## 10. Kesan yang Ingin Dicapai
Dashboard admin seharusnya terasa:
- profesional,
- modern,
- bersih,
- mudah dilihat,
